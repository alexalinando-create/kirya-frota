Política de Privacidade

Esta Política de Privacidade descreve como o aplicativo Kirya Frota coleta, usa e protege as informações dos usuários. Ao utilizar o aplicativo, você concorda com os termos descritos neste documento.

1. Informações que coletamos

O Kirya Frota coleta apenas as informações que você mesmo insere no aplicativo:

  • Dados de veículos da sua frota (placa, modelo, tipo)
  • Registros de abastecimento (quilometragem, litros, preço, data)
  • Nome dos motoristas informados nos abastecimentos
  • Dados do tanque estacionário (saldo e histórico)
  • Nome da empresa configurado nas preferências

2. Como usamos as informações

Todas as informações coletadas são utilizadas exclusivamente para:

  • Gerar relatórios e dashboards dentro do próprio aplicativo
  • Calcular custos e consumo de combustível
  • Emitir vouchers de abastecimento
  • Permitir backup e restauração dos seus dados

✓ Nenhum dado é enviado para servidores externos. Todas as informações ficam armazenadas localmente no seu dispositivo.

3. Armazenamento dos dados

Os dados do Kirya Frota são armazenados localmente no dispositivo do usuário utilizando o sistema de armazenamento seguro do Android (SharedPreferences). Não mantemos servidores próprios nem armazenamos dados em nuvem.

O usuário tem total controle sobre seus dados, podendo exportar, importar ou apagar todas as informações a qualquer momento pelas configurações do aplicativo.

4. Compartilhamento de dados

O Kirya Frota não compartilha, vende, aluga nem transfere seus dados pessoais para terceiros em nenhuma hipótese.

A única funcionalidade de compartilhamento disponível é o export de backup e relatórios em PDF, que é realizado pelo próprio usuário através dos aplicativos de sua escolha (WhatsApp, e-mail, etc.).

5. Permissões do aplicativo

O Kirya Frota solicita as seguintes permissões:

  • Bluetooth — para conexão com impressora térmica de vouchers
  • Internet — para consultar cotações de dólar e euro em tempo real
  • Armazenamento — para salvar arquivos de backup e relatórios PDF

Nenhuma permissão é utilizada para coleta de dados pessoais.

6. Segurança

O acesso ao aplicativo é protegido por login com usuário e senha definidos pelo próprio usuário. Recomendamos manter suas credenciais em segurança e não compartilhá-las com terceiros.

7. Direitos do usuário

Você tem o direito de:

  • Acessar todos os seus dados a qualquer momento
  • Exportar seus dados em formato de backup
  • Apagar todos os seus dados desinstalando o aplicativo
  • Solicitar esclarecimentos sobre esta política

8. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Recomendamos que você revise esta página ocasionalmente para se manter informado sobre quaisquer alterações. As mudanças entram em vigor imediatamente após a publicação nesta página.

9. Conformidade com a LGPD

O Kirya Frota está em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018). Como o aplicativo não coleta nem transmite dados pessoais para servidores externos, o risco de exposição de dados é mínimo.
